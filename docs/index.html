This section showcases the projects that I completed throughout the class. For each project I was provided a description of the project with the requirements to meet and an open canvas to develop each website.

Apple Website Survey
  • HTML
  • Bootstrap
  • ExpressJS
  • NodeJS

Surveys are a common method used by companies to get user feedback on their website's UI/UX. For this project I was required to create a working survey page for a chosen website in a NodeJS environment. This project gave me experience setting up a NodeJS server and using ExpressJS in the backend to handle post requests from form submissions. The front-end was built using standard html and bootstrap. Since this page runs using a localhost server it can only be viewed by forking the github repository and running it locally.

Online Grocery Store
  • HTML
  • CSS
  • JavaScript

Navigational patterns and organization of information allows users to easily navigate a webpage and identify where information can be found. This project I was required to create an smart online grocery store using a navigational pattern and organizing the products into categories. The navigational patterns used were the linear (wizard) pattern and breadcrumb pattern to simulate each step in the process of purchasing groceries. The products were sorted by category to allow for better ease of finding products for the users. The store also provided users with the options to specify their dietary needs so they would not be displayed products that are outside their diet's scope.

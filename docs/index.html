Projects

The following section showcases some of my projects completed during the class. Each project introduced a new web technology and focused its requirements on the UI concepts learned in class.

Apple Website Survey
  • HTML
  • Bootstrap
  • NodeJS

A survey to aquire user feedback related to the Apple Website UI. An express JS backend allows for form submission and data storage in a data file. The live version is a static version of th form. You can run the full version with the expressJS server by forking the GitHub repository.

live version GitHub repository
Referee Academy
  • ReactJS
  • React Bootstrap

A website to promote the Hockey Referee Academy and allow users to learn about the academy, explore ressource and sign up for a summer camp. The website was designed to cover as many heuristic values of possible to enhance the usability of the website.

live version GitHub repository
Bike Service Shop
  • HTML
  • Bootstrap
  • Jquery

An online bike service shop that allows users to explore information about the shop and book a service appointment. The UI demonstrates many design principles to improve the user's experience. Booking a service uses contraints and error prevention methods to prevent inserting potentially erroneous data.

live version
Smart Grocery Store
  • HTML
  • CSS
  • JavaScript

An online grocery store that filters product selections based on the user's specified dietary preferences. The use of navigational patterns and information organization provides the user with a simple process to purchase their groceries.

live version

Activities

This section showcases some activities completed throughout the class. Each activity was a small task that focused on the analysis and theory behind a professional User interfaces.

Personas

Two possible personas for an online conferencing software. The personas represent users of the software to allow developers to better understand the main users.

Requirements Submission
Storyboards

Storyboards for two possible personas using a recipe suggestion mobile application that makes recipe suggestions based off of the ingredients the user has available.

Requirements Submission
Knowledge Organization

A study of the knowledge organization of three university websites. Extracting the key information that each website has to offer and converting the common information into a semantic network.

Requirements Submission
Heuristic Evaluation

A heuristic evaluation of the Presto website based on ten usability heuristics to provide feedback to the developers about the areas of improvement of the websites usability.

Requirements Submission
Fonts & Colors

A study of four university websites and their methods of associating color and font to certain messages and comparing the methods to identify similarities.

Requirements Submission
